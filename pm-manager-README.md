# PM Manager — Gestão de Manutenção

## Deploy Gratuito no Vercel (2 minutos)

### Passo 1: Criar conta no GitHub (se não tem)
1. Acesse **github.com** → "Sign up"
2. Crie com seu e-mail pessoal (não use o corporativo)

### Passo 2: Criar repositório
1. No GitHub, clique **"+"** → **"New repository"**
2. Nome: `pm-manager`
3. Marque **"Public"**
4. Clique **"Create repository"**
5. Clique **"uploading an existing file"**
6. Arraste os arquivos `index.html` e `manifest.json` para a tela
7. Clique **"Commit changes"**

### Passo 3: Deploy no Vercel
1. Acesse **vercel.com** → "Sign up" com sua conta GitHub
2. Clique **"Add New..." → "Project"**
3. Selecione o repositório `pm-manager`
4. Clique **"Deploy"**
5. Pronto! Seu link será: `pm-manager-seuusuario.vercel.app`

### Passo 4: Instalar no celular (PWA)
- **Android**: Abra o link no Chrome → menu ⋮ → "Adicionar à tela inicial"
- **iPhone**: Abra no Safari → botão compartilhar → "Adicionar à Tela de Início"

O app fica com ícone na tela do celular como um app nativo!

## Funcionalidades
- Login por matrícula (CS)
- Solicitação de cadastro com aprovação
- Recuperação de senha
- Notificação de admins via WhatsApp e Teams
- Painel administrativo com PIN
- Upload de logo da empresa
- 100% independente — sem backend necessário
